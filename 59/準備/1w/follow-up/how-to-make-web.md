# html/cssで新規webサイトを作成する際の流れ

## 準備
0.フォルダを作ろう
- デスクトップでもどこでも良いので空のフォルダを作成しよう(finderで)
- 右クリックから作成できます。
- 今回のカリキュラムではNexseedフォルダを作成しているので上の操作はいりません。
- ここは今後個人的に新規でフォルダを作りたい人向けの記述です。

1.vs codeでフォルダを開こう
- vs codeを開きましょう
- Nexseedフォルダ(or 自分で0番で作成したフォルダ)をcommand + oで開きましょう

2.htmlファイル/cssファイルを作成しよう
- command + Nを押してください
- Untitled-1というファイルが作成されたと思います。
- Untitled-1というファイルにカーソルをうつして文字が入力できる状態にしてください。
- command + S を押すとfinderが開かれます。
- ファイル名を ~~.htmlとします。
- 自分が保存したい場所(Nexseedフォルダ)を選択し保存します。
- ~~ htmlファイルが出来上がりました。
- 上記の作業を繰り返して、~~.cssを作成しましょう。

3.htmlの雛形を作成しよう
- ! + tab を押して雛形を出力しよう。

4.htmlにcssファイルを読み込もう
- linkタグをheadタグの中に読み込もう。
- <link rel="stylesheet" href=""> 
- hrefの中に./と打ち込んで自分が読み込みたいcssファイルを選択します
- 階層が上の場合、../と打ち込んでみてください

## 記述
 1. htmlから書いていきましょう
 2. 大きな塊から小さい塊へと必要な要素を考えていき、htmlに記述します。
 3. cssをあてていきます。
